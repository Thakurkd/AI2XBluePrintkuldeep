# Exercise_03 — Checkboxes Feature

This folder contains a Playwright test suite for the Checkboxes feature.

Run the tests locally:

1. Install dev dependencies:

```bash
npm install
npx playwright install
```

2. Run tests:

```bash
npm test
```

Run a single spec in headed mode for debugging:

```bash
npx playwright test tests/checkboxes.spec.ts --headed
```
